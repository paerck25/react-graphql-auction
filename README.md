## React.js, Node.js, GraphQL, MongoDB 를 사용한 IT프리랜서 중개 플랫폼입니다.

디자인은 Material.ui를 사용하였습니다.

### 실행하기
1. cd client
2. npm start

### 빌드
1. cd client
2. npm run build

### docker 실행
1. cd client
2. npm run docker
2. docker run -p 4000:4000 -d paerck25/react-graphql-auction
